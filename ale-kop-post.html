O que é o Laravel?

Laravel é um framework PHP com sintaxe expressiva e elegante. Ele abstrai as tarefas mais tediosas do desenvolvimento web — autenticação, rotas, migrações — para que você possa focar no que realmente importa.

Instalar o Laravel é simples. Com o Composer configurado na sua máquina, basta rodar: 

composer create-project laravel/laravel meu-projeto
cd meu-projeto
php artisan serve

Seu servidor estará disponível em http://localhost:8000. A partir daqui, vamos explorar a estrutura de pastas e entender como cada peça se encaixa.

Estrutura de pastas
